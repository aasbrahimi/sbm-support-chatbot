# SBM Support Chatbot

# Project Overview
This project consists of a web-based support chatbot developed during an
internship at SBM.

The chatbot is designed to assist employees by answering internal support
questions using artificial intelligence.

# Objectives
- Build a functional web chatbot
- Integrate Azure OpenAI services
- Provide AI-based support for internal users
- Design a simple and user-friendly interface

# Technical Approach
- Backend implemented in PHP
- Integration with Azure OpenAI / Azure AI Foundry
- Session-based conversation history
- Secure API key handling
- Clean and minimal web interface

# Technologies
- PHP
- HTML / CSS 
- Azure OpenAI
- Azure AI Foundry

# Repository Structure
- `src/` : chatbot source code
- `guide/` : implementation guide
- `assets/` : images and UI assets
